# 💰 Finance Manager (Python Project)

## 📌 Description
A Python-based Finance Manager application to track income, expenses, and balance.

## 🚀 Features
- Add income
- Add expenses
- View balance

## 🛠️ Tech Stack
- Python
- Flask

git clone https://github.com/Bhuvaneshvari-123/Finance-Manager.git
cd Finance-Manager
python main.py
